### 3.2 EV/EBITDA倍数法 (最适合SATS)

#### 估值计算

**步骤1: 计算企业价值 (EV)**
```
市值 (中值) = $35.0B
净债务 = $29.6B - $4.3B = $25.3B
EV = $35.0B + $25.3B = $60.3B
```

**步骤2: 计算EV/EBITDA**
```
2024 EBITDA = $1.63B
EV/EBITDA = $60.3B / $1.63B = 37.0x
```

**步骤3: 行业对比**
```
行业合理EV/EBITDA倍数: 10-15x (电信/卫星行业)
SATS当前倍数: 37.0x

溢价率 = (37.0 - 12.5) / 12.5 = +196%
```

**步骤4: 合理估值计算**
```
合理EV = EBITDA $1.63B × 12.5x = $20.4B
合理股权价值 = $20.4B - $25.3B (净债务) = -$4.9B

⚠️ 警告: 负股权价值表明债务过重,股权可能接近零价值
```

**保守估值 (债务调整后)**
```
假设合理EV/EBITDA = 15x (给予5G潜力溢价)
合理EV = $1.63B × 15x = $24.5B
股权价值 = $24.5B - $25.3B = -$0.8B

⚠️ 依然为负值
```

**乐观估值 (假设债务重组/EBITDA改善)**
```
假设2026年EBITDA增长至$2.5B (基于5G业务)
合理EV = $2.5B × 15x = $37.5B
股权价值 = $37.5B - $25.3B = $12.2B
每股价值 = $12.2B / 288M = $42.36

vs 当前股价$113.22 → -63%下行空间
```
