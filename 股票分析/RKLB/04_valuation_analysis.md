## 1. 估值方法选择

### 决策树分析

```
Step 1: 是否多元化集团 (3+独立业务)?
└─ 否（两大板块高度协同）→ 进入 Step 2

Step 2: 是否成长股 + PE处于低位?
└─ PE 为负（亏损中）→ 进入 Step 3

Step 3: 自由现金流是否稳定可预测?
└─ 否（FCF 深度负值，处于高增长投入期）→ 进入 Step 4

Step 4: 默认方法
└─ 使用 相对估值法 (PS) + 远期 DCF
```

**选定估值方法组合：**

| 方法 | 权重 | 适用原因 |
|------|------|---------|
| 远期 DCF（预测盈利后） | 40% | 锚定长期价值，需预测盈利时间点 |
| PS 相对估值 | 30% | 亏损公司最常用的相对估值 |
| EV/Revenue 同业比较 | 20% | 与同行业可比公司对标 |
| 分部估值 (SOTP) | 10% | 发射+太空系统两个业务线差异化估值 |

### 2.3 WACC 计算

```
WACC = (E/(E+D)) × Re + (D/(E+D)) × Rd × (1-Tax)

其中:
- Re = Rf + Beta × ERP = 4.09% + 2.0 × 5.5% = 15.09%
- Rd = 5.0% (可转债利率)
- Tax = 0% (亏损期无实际税负)
- E/(E+D) = $37.85B / ($37.85B + $0.424B) = 98.9%
- D/(E+D) = 1.1%

WACC = 98.9% × 15.09% + 1.1% × 5.0% × 100%
     = 14.93% + 0.06%
     ≈ 15.0%
```

**WACC = 15.0%**

## 7. 安全边际分析

### 7.1 安全边际计算

```
综合目标价（基准情景）: $39.65
当前股价: $70.86

安全边际 = (内在价值 - 当前价) / 内在价值
        = ($39.65 - $70.86) / $39.65
        = -78.7%

→ 当前股价高于估算内在价值约 79%
→ 安全边际为负，即当前价格不存在安全边际
```

### 7.2 赔率分析

```
上行空间 (乐观 $85): +$14.14 (+20.0%)
下行空间 (悲观 $30): -$40.86 (-57.7%)

赔率 = 上行空间 / 下行空间 = 20.0% / 57.7% = 0.35

赔率 < 1: 风险收益比不利
```

## 9. 估值结论

### 9.1 核心发现

1. **传统 DCF 显示显著高估**：加权 DCF 目标价 $3.85 vs 当前 $70.86，差距 18 倍
2. **相对估值（PS/EV Revenue）更贴近市场**：基准 $60-67，接近当前价格
3. **SOTP 显示中等高估**：$25.84 vs $70.86，高估约 174%
4. **市场定价主要驱动力**：Neutron 首飞预期 + 叙事驱动 + 太空赛道热度

### 9.2 估值判断

| 判断维度 | 结论 |
|----------|------|
| 内在价值 (DCF) | 严重高估 ($3-9 vs $70.86) |
| 相对价值 (PS) | 接近合理但偏高 (PS 68x vs 行业 25-40x) |
| 安全边际 | **不存在** (负安全边际) |
| 赔率 | **不利** (0.35, 下行空间远大于上行) |
| 市场情绪 | 极度乐观 |

### 9.3 关键估值风险

1. **如果 Neutron 首飞失败**：股价可能跌至 $25-35 区间 (-50%~-65%)
2. **如果盈利时间线推迟至 2030 年**：DCF 价值进一步下降
3. **如果太空行业热度消退**：PS 倍数可能从 65x 压缩至 20-30x
4. **如果 SpaceX Starship 完全商业化**：Neutron 的市场空间可能大幅缩小
