## 1. 业务基本盘

### 1.1 公司概况

### 1.2 核心业务

Rocket Lab 是一家**全栈式太空公司（Full-Stack Space Company）**，业务涵盖从火箭研发制造、卫星发射服务到航天器组件和整星制造：

**两大业务板块：**

#### (1) 发射服务（Launch Services）
- **Electron 火箭**：小型运载火箭，低地球轨道（LEO）有效载荷约 300kg
  - 截至2025年底已完成 50+ 次发射
  - 全球第二高发射频率（仅次于 SpaceX）
  - 2025年完成约 16 次发射，创历史新高
  - 合同在手发射任务：49 次
  - 单次发射成本：约 $7.5M-$8M
- **Neutron 火箭**（开发中）：中型可复用运载火箭
  - LEO 有效载荷：约 13,000 kg
  - 目标竞争对手：SpaceX Falcon 9
  - 首飞时间：2026年中
  - 2026年计划发射 3 次，2027年计划 5 次
  - 可复用设计，大幅降低单次发射成本

#### (2) 太空系统（Space Systems）
- **卫星制造**：为政府和商业客户设计制造整星
  - SDA（太空发展局）$816M 合同：18颗导弹跟踪卫星
  - MDA（导弹防御局）等政府项目
- **航天器组件**：
  - Photon 卫星平台
  - 太阳能板、反应轮、星跟踪器
  - Rutherford 发动机、Archimedes 发动机（Neutron用）
- **在轨管理与任务控制**

### 1.3 收入结构

| 业务板块 | Q3 2025 收入占比 | YoY 增长 | 订单积压占比 |
|----------|-----------------|----------|-------------|
| 发射服务 | ~35% | +94.8% | 47% |
| 太空系统 | ~65% | +36.1% | 53% |

- 太空系统占比持续提升，体现公司从"火箭公司"向"全栈太空公司"的转型
- 发射服务增速更快（+94.8% YoY），但受客户卫星交付延迟影响波动较大
- 总订单积压（Backlog）：**$1.4B**（$816M SDA 合同后）

### 1.4 商业模式

**"端到端太空解决方案"模式：**
1. **重复性收入**：发射服务具有重复性特征，每次 Electron 发射 $7.5-8M
2. **长周期政府合同**：SDA $816M 合同贡献 5 年收入可见性
3. **垂直整合**：自研发动机、航天器组件降低成本
4. **平台效应**：发射服务 + 太空系统形成闭环，客户粘性强
5. **技术复用**：Electron 积累的技术经验加速 Neutron 开发

### 1.5 目标客户

- **政府/军事**：美国太空军（USSF）、SDA、NASA、NRO、DARPA
- **商业客户**：卫星星座运营商、通信公司、遥感公司
- **国际客户**：日本、韩国、欧洲航天客户

## 2. 护城河评估（巴菲特 5 大护城河）

### 2.1 无形资产（品牌/专利/许可）⭐⭐⭐⭐ (4/5)

| 维度 | 评估 |
|------|------|
| 品牌认知 | 全球第二大商业火箭发射公司，"Electron"品牌在小型火箭领域几乎等同于品类名称 |
| 专利/技术 | 3D打印 Rutherford 发动机技术（全球首个电泵循环发动机）；碳复合材料火箭结构技术；Photon 卫星平台技术 |
| 许可/资质 | FAA 商业发射许可证；美国国防部安全许可；新西兰发射场许可 |
| 政府信任 | SDA 合同中标体现高度政府信任；多次执行 NRO、DARPA 等机密任务 |

**评级：强**
- 核心技术壁垒高：3D打印发动机和碳复合材料结构是独有技术
- 政府安全许可和信任关系难以复制
- 50+ 次发射的飞行传承（Flight Heritage）是强大背书

### 2.2 转换成本 ⭐⭐⭐ (3/5)

| 维度 | 评估 |
|------|------|
| 客户锁定 | 发射服务本身转换成本较低（客户可选择其他火箭）；但整星制造合同通常多年锁定 |
| 技术整合 | 使用 Photon 平台的客户迁移到其他平台成本高 |
| 合同粘性 | SDA $816M 合同锁定至 2029 年 |

**评级：中等**
- 发射服务竞争激烈，转换成本相对较低
- 太空系统业务（卫星制造）转换成本较高
- 长期政府合同提供了一定的锁定效应

### 2.3 网络效应 ⭐⭐ (2/5)

**评级：弱**
- 太空发射行业不具备典型的网络效应
- 但"全栈"模式创造了一定的生态效应：发射客户可能成为卫星制造客户
- Electron 的高频发射积累的可靠性数据，间接增强了客户信心

### 2.4 成本优势 ⭐⭐⭐⭐ (4/5)

| 维度 | 评估 |
|------|------|
| 垂直整合 | 自研发动机、电子系统、航天器组件，减少供应链依赖 |
| 制造创新 | 3D打印 Rutherford 发动机大幅降低制造成本和时间 |
| 规模效应 | 高频发射摊薄固定成本；Electron 年发射 16+ 次 |
| Neutron 设计 | 可复用设计将大幅降低中型火箭发射成本 |

**评级：强**
- 垂直整合 + 创新制造工艺提供显著成本优势
- Neutron 的可复用设计是关键成本优势来源
- 相对 ULA、Arianespace 等传统竞争对手优势明显

### 2.5 规模优势 ⭐⭐⭐ (3/5)

| 维度 | 评估 |
|------|------|
| 市场份额 | 小型火箭领域 #1（唯一频繁商业发射的竞争者） |
| 发射频率 | 全球第二（仅次于 SpaceX），2025年 ~16 次发射 |
| 基础设施 | 3 个发射场（新西兰 LC-1A/1B、弗吉尼亚 LC-2、Neutron LC-3） |
| 全球覆盖 | 北半球+南半球发射能力 |

**评级：中等偏强**
- 在小型火箭领域具有规模领先优势
- 但与 SpaceX 相比仍然有量级差距
- Neutron 进入中型火箭市场后将面对 Falcon 9 的规模壁垒

### 护城河综合评级：**中等偏宽（Moderate-Wide）** ⭐⭐⭐⭐ (7.5/10)

**核心护城河来源**：
1. 技术壁垒（3D打印发动机、碳复合材料、50+飞行传承）
2. 垂直整合带来的成本优势
3. 政府安全许可和信任关系
4. 小型火箭领域的规模领先

**护城河风险**：
- SpaceX 的绝对主导地位（95% 美国发射份额）
- 新进入者（Blue Origin、Relativity Space）的竞争威胁
- 技术路线变化风险（可复用技术已成标配）

## 3. 竞争格局

### 3.1 主要竞争对手

| 竞争对手 | 火箭产品 | LEO载荷 | 可复用 | 发射频率(2025) | 状态 |
|----------|---------|---------|--------|---------------|------|
| **SpaceX** | Falcon 9/Heavy, Starship | 22,800kg / 150,000kg | 是 | 130+ 次 | 绝对主导 |
| **ULA** | Vulcan Centaur | 27,200kg | 部分 | ~6 次 | 转型中 |
| **Blue Origin** | New Glenn | 45,000kg | 部分 | 首飞2025 | 初期 |
| **Arianespace** | Ariane 6 | 21,650kg | 否 | ~4 次 | 重启中 |
| **Relativity Space** | Terran R | 23,500kg | 是 | 开发中 | 未首飞 |
| **Rocket Lab** | Electron / Neutron | 300kg / 13,000kg | Neutron是 | 16 次 | #2位置 |

### 3.2 竞争定位

```
                    高载荷
                      ↑
         ULA          |  SpaceX (Falcon 9/Heavy)
         Blue Origin  |
                      |
    ← 低频率 ─────────┼──────── 高频率 →
                      |
                      |  Rocket Lab (Neutron 目标区)
         Relativity   |
                      |  Rocket Lab (Electron)
                      ↓
                    低载荷
```

### 3.3 市场份额

- **SpaceX**：美国发射市场 ~95% 份额，全球 ~65%
- **Rocket Lab**：小型火箭市场 #1，全球发射市场 ~3-5%
- **全球发射服务市场规模**：$11.9B (2025)，预计 2029 年达 $22.18B（CAGR ~17%）
- **每公斤发射成本**：
  - Falcon 9: ~$2,720/kg
  - Electron: ~$19,000/kg（但小型专属发射，不共享轨道）
  - Neutron（目标）: ~$5,000-6,000/kg

### 3.4 竞争优势总结

**Rocket Lab 的差异化优势：**
1. **唯一与 SpaceX 并列的频繁发射商**（全球 #2）
2. **全栈能力**：从组件到整星到发射的一站式服务
3. **双发射场覆盖**：北半球+南半球，轨道覆盖广
4. **政府合同多样化**：SDA、NRO、DARPA、NASA 多方信任
5. **从小到大的发展路径**：Electron 验证技术 → Neutron 扩展市场

## 4. 行业分析

### 4.1 行业生命周期

**商业太空行业处于「成长期」中段**

| 特征 | 当前状态 |
|------|---------|
| 市场增速 | 年均 15-20%，高增长 |
| 新进入者 | 大量新公司，但存活率低 |
| 技术迭代 | 可复用火箭成为标配，降低进入门槛 |
| 客户需求 | 卫星星座（LEO 通信、遥感、国防）驱动需求爆发 |
| 利润水平 | 头部公司开始盈利，多数仍亏损 |
| 政府投入 | 大幅增加（美国太空军、SDA、NASA Artemis） |

### 4.2 TAM（可服务市场）

| 市场维度 | 2025 规模 | 2029 预测 | CAGR |
|----------|----------|----------|------|
| 全球发射服务市场 | $11.9B | $22.18B | ~17% |
| 小型卫星发射市场 | $3.5B | $7.0B | ~19% |
| 太空系统/卫星制造 | $15B+ | $25B+ | ~14% |
| Rocket Lab 可寻址市场 | ~$10B | ~$20B | ~19% |

### 4.3 行业壁垒

| 壁垒类型 | 强度 | 说明 |
|----------|------|------|
| 资金壁垒 | 极高 | 火箭研发需 $1B+ 投入，Neutron 研发预算约 $300M |
| 技术壁垒 | 极高 | 火箭科学复杂，发动机、制导、再入技术难度大 |
| 监管壁垒 | 高 | FAA 许可、ITAR 出口管制、国防安全许可 |
| 经验壁垒 | 高 | 飞行传承（Flight Heritage）需要多年积累 |
| 客户信任 | 高 | 尤其政府/国防客户，建立信任需要长期合作记录 |

### 4.4 行业驱动力

1. **LEO 卫星星座建设**：Starlink、OneWeb、Amazon Kuiper 等驱动大量发射需求
2. **太空军事化**：美国太空军/SDA 大规模采购卫星和发射服务
3. **商业太空旅游**：高净值市场（SpaceX、Blue Origin）
4. **太空经济数据化**：遥感、气象、导航等应用场景扩大
5. **小型卫星革命**：体积小、成本低、部署快的小卫星成为主流趋势

## 6. 长期趋势（3-5年展望）

### 6.1 增长机遇

1. **Neutron 火箭商业化**（2026-2028）
   - 进入 $10B+ 的中型火箭市场
   - 可复用设计降低发射成本至 $5,000-6,000/kg
   - 目标成为 Falcon 9 之外的第二选择
   - 2026 计划 3 次发射，2027 计划 5 次

2. **政府/国防合同增长**
   - SDA $816M 合同是起点，未来 Tranche 4/5 更多机会
   - 美国太空军年度预算持续增长
   - 北约盟国的太空防御需求上升

3. **太空系统业务规模化**
   - 从组件供应商升级为整星制造商
   - 垂直整合带来更高利润率
   - 卫星星座批量制造能力

4. **发射频率提升**
   - Electron 年发射目标 20+ 次
   - Neutron 加入后总发射次数大幅增长
   - 发射基础设施已就绪

5. **国际市场拓展**
   - 亚太市场（日本、韩国）
   - 欧洲发射服务空缺（Ariane 6 产能不足）

### 6.2 关键风险

1. **Neutron 首飞风险**：火箭首飞失败将严重打击市场信心和估值
2. **持续亏损与融资风险**：可能需要继续融资稀释股东
3. **SpaceX 竞争压力**：Starship 成功将进一步扩大 SpaceX 优势
4. **客户集中度**：政府合同占比高，政策变化可能影响收入
5. **关键人风险**：Peter Beck 是公司灵魂，其去留对公司影响巨大
6. **估值泡沫风险**：当前 PS ~63x，远超行业平均
