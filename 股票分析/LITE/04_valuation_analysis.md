## 1. DCF 绝对估值

### 1.1 核心假设参数

**宏观参数**:
- 无风险利率 (Rf): 4.09% (美国10年期国债)
- 市场风险溢价 (MRP): 5.0%
- Beta系数: 1.47 (取1.40和1.53的平均值)
- 公司所得税率: 15% (估计有效税率)

**WACC 计算**:
```
股权成本 (Re) = Rf + Beta × MRP
              = 4.09% + 1.47 × 5.0%
              = 11.44%

E (市值) = $47,700M
D (净债务) = $2,087M
E/(E+D) = 95.8%
D/(E+D) = 4.2%

Rd (债务成本) = 3.0% (可转债票面利率较低)
Tax = 15%

WACC = 95.8% × 11.44% + 4.2% × 3.0% × (1 - 15%)
     = 10.96% + 0.11%
     = 11.07%

→ WACC ≈ 11.0%
```

**FY2026基年预测** (公司财年7月-6月):
```
Q1 FY2026 (实际): 营收 $533.8M
Q2 FY2026 (实际): 营收 $665.5M
Q3 FY2026 (指引): 营收 $805M (中值)
Q4 FY2026 (估计): 营收 $900M

FY2026E 全年营收: ~$2,904M
FY2026E Non-GAAP 营业利润率: ~27%
FY2026E Non-GAAP 营业利润: ~$784M
FY2026E CapEx: ~$350M
FY2026E D&A: ~$270M
FY2026E FCF估计: NOPAT + D&A - CapEx - WC变动
                = $666M + $270M - $350M - $50M

```
