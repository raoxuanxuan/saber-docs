# 【LITE】Lumentum Holdings Inc. - 财务分析报告

## 执行摘要

Lumentum正经历一次财务指标的剧烈改善：从FY2024的GAAP净亏损$5.47亿快速转向FY2026H1的盈利，毛利率从23.7%回升至39%，Non-GAAP营业利润率达到25.2%。但GAAP与Non-GAAP之间存在巨大差异（FY2025 SBC $1.77亿），自由现金流连续两年为负，且高达$26亿的长期债务是需要密切关注的风险。财务指标正在快速改善，但尚未完全恢复到FY2021-2022的峰值水平。

## 6. 财务分析结论

**财务优势** (Top 3):
1. **爆发性增长**: 营收TTM+49%，Q2 FY2026 +65.5% YoY，Q3指引继续加速；Non-GAAP EPS从$0.30快速增至$1.67
2. **利润率快速修复**: Non-GAAP毛利率从30.2%回升至42.5%，营业利润率从近零恢复至25.2%，趋向目标30%+
3. **强运营杠杆**: 收入每增长1%带动毛利润增长约1.75%，规模效应显著

**财务风险** (Top 3):
1. **自由现金流持续为负**: FCF连续两年为负，TTM仍为-$29.5M，大规模CapEx消耗现金
2. **高负债率**: 资产负债率73.1%，净债务/EBITDA ~5.3x，主要为可转债但仍需关注
3. **GAAP与Non-GAAP差异巨大**: FY2025 SBC $1.77亿+折旧$2.54亿，GAAP盈利能力被严重拖累

**财务质量判断**:
- [x] **一般财务 → 快速改善中** — 增长指标优秀，但FCF为负、负债率偏高、盈利质量需持续验证
- 预计FY2026下半年财务指标将显著改善（FCF转正、利润率继续扩张）
- 关键观察点: Q3 FY2026实际结果能否达到$780-830M指引

**⚠️ 估值相关警示**: 当前市值$477亿 / TTM营收$21亿 = PS 22.7x；市值$477亿 / TTM净利润$2.52亿 = PE 189x。即使FY2026全年Non-GAAP EPS达到$7-8，Forward PE仍高达84-96x。财务分析显示基本面改善明确，但估值合理性需Agent 04进一步评估。
