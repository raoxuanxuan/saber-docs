## 1. HBM 行业概览

### 1.1 什么是 HBM

HBM（High Bandwidth Memory，高带宽内存）是专为 AI 加速芯片设计的高性能内存，通过 3D 堆叠技术将多层 DRAM 芯片垂直叠加，以极高带宽满足 GPU/ASIC 的数据吞吐需求。

### 1.2 HBM 与普通内存的核心差异

| 对比项 | 普通 DDR5 | HBM3E |
|--------|---------|-------|
| 带宽 | ~100 GB/s | **3,000+ GB/s** |
| 容量（单条/栈） | 16-64 GB | 24-36 GB/栈 |
| 功耗效率 | 一般 | 极优（近距离传输） |
| 价格溢价 | 基准 | **5-8x** |
| 制造难度 | 低 | 极高（TSV 堆叠） |
| 应用场景 | 通用计算 | AI 训练/推理专用 |

### 1.3 HBM 技术代际路线

| 规格 | 带宽/栈 | 容量/栈 | 主要应用 | 状态 |
|------|--------|--------|---------|------|
| HBM2E | 460 GB/s | 16 GB | A100 早期 | 退出主流 |
| HBM3 | 819 GB/s | 16-24 GB | H100 | 尾声 |
| **HBM3E** | **1.2 TB/s** | **24-36 GB** | **H200/B200/Ironwood** | **当前主流** |
| **HBM4** | **>2.6 TB/s** | **36-48 GB** | **Rubin R200** | **2026 量产** |
| HBM4E | ~4 TB/s | 48 GB+ | Rubin Ultra | 2027 |

### 1.4 HBM 市场规模

| 年份 | 市场规模 | YoY |
|------|---------|-----|
| 2023 | ~$4B | — |
| 2024 | ~$18B | +350% |
| 2025 | **~$35B** | +94% |
| 2026E | ~$44B | +26% |
| 2028E | **~$100B** | CAGR ~40%（美光预测） |

### 1.5 当前市场份额分布（2025 Q2）

## 7. 为什么更贵的 HBM 反而降低推理成本

### 7.1 核心逻辑

```
推理成本/token = 硬件总成本 ÷ 每秒处理的 token 数

HBM4 更贵    → 分子↑
性能提升更多  → 分母↑↑↑（5x 以上）

结果：每 token 成本反而下降 10x（英伟达官方数据）
```

### 7.2 英伟达官方对比数据（Blackwell → Rubin）

| 指标 | 变化 |
|------|------|
| 芯片价格 | ↑（HBM4 更贵） |
| 推理算力（NVFP4） | **↑ 5x**（50 PFLOPS） |
| **每 token 推理成本** | **↓ 10x** |
| 训练 MoE 所需 GPU 数 | **↓ 4x** |

### 7.3 五大技术原因

**① 解决内存带宽瓶颈**（最核心）

大模型推理时 GPU 往往等内存传数据（memory-bound）：
```
Blackwell: 8 TB/s  → GPU 经常空等
Rubin:    13 TB/s  → 等待减少，GPU 利用率大幅提升
```

**② 显存容量翻倍 → 减少跨卡通信**

| 芯片 | 显存 | 效果 |
|------|------|------|
| B200 | 192 GB | 大模型需切片 |
| R200 | 288 GB | 更大模型单卡承载 |
| Rubin Ultra | 1 TB | 超大模型单机运行 |

**③ MoE 架构专项优化**：针对 GPT-4、Gemini、DeepSeek 等主流架构。

**④ 算力 5x 提升**：50 PFLOPS vs Blackwell 10 PFLOPS。

**⑤ 系统级规模经济**：替换多台旧 GPU，总功耗和机架成本下降。

### 7.4 对 HBM 的飞轮效应

```
推理成本↓ → AI 应用更便宜 → 使用量爆发
  → 更多 GPU 需求 → 更多 HBM 需求
  → 三家厂商全部受益
```

```

长期：HBM4/HBM4E 等高端产品仍是三巨头专属
美国出口管制 = 三巨头的额外护城河
```
