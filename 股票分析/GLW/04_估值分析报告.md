## 1. DCF 绝对估值

### 1.1 核心假设

#### WACC 计算

```
WACC = (E / (E + D)) x Re + (D / (E + D)) x Rd x (1 - Tax)

其中:
E = 市值 = $119.69B
D = 净债务 = $7.85B
Re = 股权成本 = Rf + Beta x MRP = 4.09% + 1.12 x 5.5% = 10.25%
Rd = 债务成本 = 利息支出 / 总债务 = $0.34B / $9.38B = 3.62%
Tax = 实际税率 = $0.31B / $2.05B = 15.1%
```

**计算过程**:
```
E / (E + D) = $119.69B / ($119.69B + $7.85B) = 93.8%
D / (E + D) = $7.85B / ($119.69B + $7.85B) = 6.2%

WACC = 93.8% x 10.25% + 6.2% x 3.62% x (1 - 15.1%)
     = 9.61% + 0.19%
     = 9.80%
```

**WACC = 9.80%**（取整 ~10.0%）

| 参数 | 数值 | 来源 |
|------|------|------|
| 无风险利率 (Rf) | 4.09% | 美国 10 年期国债收益率 (2026-02-20) |
| 市场风险溢价 (MRP) | 5.5% | 美股历史均值 |
| Beta | 1.12 | Yahoo Finance |
| 股权成本 (Re) | 10.25% | CAPM 计算 |
| 债务成本 (Rd) | 3.62% | 利息支出/总债务 |
| 有效税率 | 15.1% | FY2025 |
| **WACC** | **9.80%** | 加权计算 |

| Meta 合约规模缩减 50% | $60 亿 → $30 亿 | 综合目标价 -15% 至 $95.78 | **中** |

**结论**: 综合估值对光通信增速和净利率最为敏感。若 AI 光通信增速低于 25% 且净利率未能扩张，综合目标价可能降至 $95-100 区间。

## 10. 估值结论

### 核心发现

1. **DCF 内在价值**: $32.47（基准）/ $34.29（概率加权）-- 远低于当前价，反映 FCF 不足以支撑市值

3. **戴维斯双击 3 年目标价**: $155.02（概率加权，年化 +3.6%）-- 唯一接近当前价的方法
4. **相对估值目标价**: $88.55 - $145.15 区间，中枢约 $110
5. **华尔街共识目标价**: $118.00（13 位分析师，低于现价 15.4%）
6. **综合目标价**: **$112.68**（下行空间 -19.2%）

### 投资亮点

1. **AI 光纤需求的确定性极高**: Meta $60 亿合约、光纤密度 16x 跃升、多个超大规模客户扩产，增长能见度 3-5 年

3. **盈利拐点已确认**: FY2025 净利润 +214%，运营杠杆充分释放

### 关键风险

1. **估值风险（最高优先级）**: 综合目标价 $112.68 vs 当前价 $139.51，存在 19.2% 下行空间。若增速不及预期，PE 压缩 + EPS 失望的"双杀"可能导致 50%+ 回调

4. **估值对净利率最敏感**: 若净利率未能从 10.2% 提升至 14%+，Forward PE 的"E"将无法兑现
