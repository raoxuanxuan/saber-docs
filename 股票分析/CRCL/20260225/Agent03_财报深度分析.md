## 执行摘要

- **GAAP 净亏损 $7,000 万**（受 $4.24 亿 IPO 相关非现金股权激励费用影响）

- **盈利质量评级：B-（中等偏上）**——现金流质量良好，但过度依赖外部利率环境

**盈利质量综合评分：62/100**

### 1.3 成本结构深度分析

**最关键的成本：Coinbase 分成协议**

```
FY2024 Coinbase 分成结构（最新公开数据）：
• Coinbase 托管的 USDC：约 20% × $43.7B = $8.74B
• $8.74B × 4.7% 储备率 = $411M → 100% 归 Coinbase
• 剩余储备收入 $1,250M × 50% = $625M → 归 Coinbase
• Coinbase 总分成：$411M + $497M = $908M（约占总储备收入 56%）
• Circle 净留存：$1,676M - $908M - 其他分发 = 约 $768M（net）
```

**FY2025 估算（Coinbase 份额升至 22%）**：
- Coinbase 托管 USDC：22% × $55B（平均）= $12.1B × 4.0% = $484M（100%）
- 剩余 $2,157M × 50% = $1,079M
- Coinbase 估算分成：$484M + $1,079M = $1,563M
- Circle 净留存（仅储备收入）：约 $1,078M

**⚠️ 核心风险**：Coinbase 比例从 2024 年 20% 增至 2025 年 22%，若升至 30%，Circle 毛利率将进一步压缩至接近盈亏平衡。

## 5. Altman Z-Score 分析

**⚠️ 特殊说明**：传统 Z-Score 使用总资产作为分母，但 Circle 总资产 $78.7B 中 96% 是 USDC 储备（客户资产），直接使用会导致误导性结论。本报告使用**调整后资产（剔除 USDC 透传）**。

| Z-Score 因子 | 计算 | 数值 | 权重 | 加权值 |
|------------|------|------|------|-------|
| X1 = 营运资本/总资产 | $1,330M / $3,400M | 0.391 | 1.2 | 0.470 |
| X2 = 留存收益/总资产 | $~-200M / $3,400M | -0.059 | 1.4 | -0.082 |
| X3 = EBIT/总资产 | -$96M / $3,400M | -0.028 | 3.3 | -0.093 |
| X4 = 股权市值/总负债 | $17,140M / $137M | 125.1 | 0.6 | 75.06 |
| X5 = 营收/总资产 | $2,747M / $3,400M | 0.808 | 1.0 | 0.808 |
| **Z-Score 合计** | | | | **76.2** |

**Z-Score 解读（调整后）**：76.2 >> 2.99（安全区），得益于市值远超实际债务。

**若使用传统未调整 Z-Score（含 USDC）**：
- X4 = $17,140M / $75,382M = 0.227 → 加权 0.136
- 其他因子因分母膨胀大幅缩小
- Z-Score 约为 0.5（进入"危险区"）——**但这是方法论的局限，非真实财务风险**

**实质性破产风险评估**：极低——有息债务仅 $37M，现金 $1.53B，Adj.EBITDA $582M，债务覆盖率 15,730%。
