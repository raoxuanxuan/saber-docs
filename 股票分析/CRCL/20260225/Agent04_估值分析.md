## 2. Altman Z-Score（破产风险检查）

**调整方法**：剔除 USDC 储备资产（客户资产）和对应负债后计算

| 因子 | 计算 | 数值 |
|------|------|------|
| X1 营运资本/总资产 | $1,330M / $3,400M | 0.391 |
| X2 留存收益/总资产 | -$200M / $3,400M | -0.059 |
| X3 EBIT/总资产 | -$96M / $3,400M | -0.028 |
| X4 权益市值/总负债（实际）| $17,140M / $137M | 125.1 |
| X5 营收/总资产 | $2,747M / $3,400M | 0.808 |

**Z-Score = 1.2(0.391) + 1.4(-0.059) + 3.3(-0.028) + 0.6(125.1) + 1.0(0.808)**
**= 0.47 - 0.08 - 0.09 + 75.06 + 0.81 = 76.2**

**Z-Score 评级**：76.2 >> 2.99（安全线）→ **破产风险极低**

**实质原因**：有息负债仅 $37M，市值 $17.14B，市值/负债 = 125:1

**→ 通过 Gate 2 破产风险检查**

## 6. 逆向 DCF（市场隐含假设）

**问题**：当前股价 $72.81 隐含了什么预期？

**逆向求解**（WACC 13%, g=4%）：
- 当前 EV：$15.65B
- 若每股 FCF 年化值 × (1/(WACC-g)) = 15.65B
- 隐含终值 FCF：$15.65B × (0.13-0.04) = $1,408M → 在 WACC=13%、g=4% 下
- 这需要 5 年后 FCF 约 $1,200-1,400M

**验证可行性**：
- FY2025 Adj.EBITDA: $582M → FCF ~$520M
- FCF 需增长至 $1,200-1,400M（+130-170%），约 5 年 18-22% CAGR
- 对应 USDC 维持 30-35% CAGR + 毛利率改善
- **结论**：这是可实现但需要执行的预期，不是太离谱的假设

**市场隐含 USDC 增速**：约 33-38% CAGR（高于基准估计的 30%，低于公司指引的 40%）
→ **市场定价大致合理，轻微乐观偏置**
