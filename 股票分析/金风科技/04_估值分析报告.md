## 1. Altman Z-Score 破产风险检查

### 1.1 Z-Score公式（制造业版本）

**Z = 1.2 x X1 + 1.4 x X2 + 3.3 x X3 + 0.6 x X4 + 1.0 x X5**

### 1.2 各因子计算（基于2024年年报数据，单位：百万人民币）

**基础数据**：
- 总资产（TA）：155,224
- 流动资产（CA）：68,601
- 流动负债（CL）：71,874
- 总负债（TL）：114,797
- 营业收入（Rev）：56,699
- EBIT：1,538
- 留存收益（RE）：估算值，归母净资产40,427 - 股本约4,227 - 资本公积约12,000 = ~24,200（估）
- 市值（H股+A股综合）：A股市值约1,141亿人民币 = 114,100；H股部分以A股价格计（因同一公司）

**X1 = 营运资本 / 总资产**
```
营运资本 = CA - CL = 68,601 - 71,874 = -3,273
X1 = -3,273 / 155,224 = -0.0211
```

**X2 = 留存收益 / 总资产**
```
X2 = 24,200 / 155,224 = 0.1559
```

**X3 = EBIT / 总资产**
```
X3 = 1,538 / 155,224 = 0.0099
```

**X4 = 股权市值 / 总负债账面值**
```
股权市值（以A股市值为基准） = 114,100
X4 = 114,100 / 114,797 = 0.9939
```

**X5 = 营收 / 总资产**
```
X5 = 56,699 / 155,224 = 0.3653
```

### 1.3 Z-Score计算

```
Z = 1.2 x (-0.0211) + 1.4 x 0.1559 + 3.3 x 0.0099 + 0.6 x 0.9939 + 1.0 x 0.3653

  = (-0.0253) + 0.2183 + 0.0327 + 0.5963 + 0.3653

Z = 1.1873
```

### 1.4 Z-Score判定

| 区间 | 判定 | 金风科技 |
|------|------|---------|
| Z > 2.99 | 安全区 | -- |
| 1.81 < Z < 2.99 | 灰色区 | -- |
| **Z < 1.81** | **危险区** | **Z = 1.19** |

**结论：Z-Score = 1.19，处于"危险区"（Distress Zone），表明公司存在较高的财务困境风险。**

**主要拖累因素**：
1. **X1为负值（-0.021）**：流动负债超过流动资产，营运资本为负
2. **X3极低（0.010）**：EBIT/总资产仅1%，资产盈利能力极弱
3. **X4接近1（0.994）**：市值仅略超过总负债，安全边际不足

**风险评估**：虽然Z-Score处于危险区，但需要注意以下缓解因素：
- 金风科技是风电场重资产运营商，类似于电力公用事业公司，高负债率有行业特殊性
- 公司主要负债为风电场项目贷款，有对应的发电资产和长期购电协议（PPA）作为支撑
- 机构预测2025年净利润将大幅改善至35.9亿元，X3指标将显著提升
- 但该Z-Score提醒我们，在估值中必须充分考虑财务风险溢价

### 2.6 DCF模型局限性说明

本DCF分析存在以下重要局限：

1. **FCF连续为负的困境**：传统DCF依赖正FCF折现，金风科技连续5年负FCF使传统方法失效。本报告采用"稳态盈利折现法"作为替代，但引入了终端PE假设，降低了模型的纯粹性。

2. **高杠杆放大估值波动**：净债务270亿元对EV到股权价值的转换产生巨大影响。净债务减少/增加50亿元将导致每股价值变化约1.3港元。

3. **营运资本变化假设的敏感性**：应收账款和存货的变化趋势对FCF预测有决定性影响，但预测准确性较低。

### 7.2 安全边际计算

```
当前H股价格：~13.50港元
综合内在价值：~16.80港元

安全边际 = (内在价值 - 当前价格) / 内在价值
         = (16.80 - 13.50) / 16.80
         = 19.6%
```

**安全边际：19.6%** -- 低于理想的30%安全边际要求，但考虑到D+财务评级，建议更高的安全边际门槛。

### 7.3 赔率（Odds Ratio）计算

| 情景 | 目标价（港元） | 相对当前价涨跌幅 | 概率 |
|------|-------------|----------------|------|
| 乐观上行 | 29.02 | +115% | 25% |
| 基准情景 | 16.76 | +24% | 50% |
| 悲观下行 | 8.38 | -38% | 25% |

```
期望收益 = 25% x 115% + 50% x 24% + 25% x (-38%)
         = 28.75% + 12.0% + (-9.5%)
         = +31.25%

上行空间（加权） = 25% x 15.52 + 50% x 3.26 = 5.51港元
下行空间（加权） = 25% x 5.12 = 1.28港元

赔率 = 上行空间 / 下行空间 = 5.51 / 1.28 = 4.30x
```

**赔率：4.30x** -- 赔率大于3x，表明风险收益比较为有利。

### 7.4 投资建议

**综合评级：谨慎买入（Cautious Buy）**

**核心逻辑**：

1. **估值吸引力中等偏上**：H股2026E PE约10.9x，处于历史低位区间和国内同行折扣水平，相对估值具有一定吸引力

2. **安全边际不足**：19.6%的安全边际低于30%的理想标准，考虑到D+级财务评级，投资者应要求更高的安全边际

3. **赔率较优**：4.30x的赔率反映了乐观情景的显著上行空间（蓝箭航天IPO + 海外突破 + 毛利率回升的三重催化）

4. **风险因素不可忽视**：
   - Z-Score处于危险区（1.19），财务困境风险真实存在
   - 连续5年负FCF，资金依赖外部融资
   - 利息覆盖倍数仅1.26x，偿债压力大
   - 欧盟调查可能影响海外拓展

5. **建议操作**：
   - **激进投资者**：当前价位（~13.50港元）可开始建仓，仓位控制在5%以内
   - **稳健投资者**：等待股价回调至12港元以下（安全边际>28%）时分批买入
   - **保守投资者**：暂不建仓，等待2025年年报确认盈利改善和现金流转正的信号
   - **止损位**：建议设定在10.0港元（较当前价约-26%）
