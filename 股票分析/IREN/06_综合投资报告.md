## 三、Bull/Bear Case 深度分析

### 3.1 Bull Case (乐观情景)

**核心论点**: IREN是AI基础设施超级周期的最大受益者之一，电力稀缺+Microsoft合同将驱动公司从$0.5B收入成长至$50-80B年化收入的平台型企业。

**支撑要素**:

1. **AI基础设施需求爆炸**: 2026年全球超大规模云商资本开支预计>$6,500亿，其中75%投向AI基础设施。电力（而非资本）是最大瓶颈，IREN的4.5GW+储备是极度稀缺资源。

2. **Microsoft合同验证**: $9.7B五年期合同 + $1.9B预付款 + $3.6B低息GPU融资 = 收入确定性极高、融资压力大幅缓解。仅此合同年化即可产生$19.4亿收入。

3. **估值框架切换**: 一旦AI收入超过BTC挖矿收入（预计FY2027-2028），市场将用AI基础设施估值框架（PS 15-25x）替代矿企框架（PS 5-8x），触发显著的估值倍数扩张。

4. **新客户拓展潜力**: Microsoft合同验证了IREN的能力，有望吸引Google、Meta、Oracle等更多超大规模客户，$11亿待签ARR空间巨大。

5. **Fed降息预期**: 若2026年Q2-Q3降息，将降低$3.6B GPU融资成本并提振高成长/高杠杆公司估值。

**Bull Case估值**: FY2029E 营收$6.0B x 净利率15% = 净利润$0.90B，EPS ~$2.25（400M稀释后），给予30x PE = **$67.50/股**。DCF乐观情景: **$73.58/股**。

### 3.2 Bear Case (悲观情景)

**核心论点**: IREN是一场高风险赌注，BTC熊市+执行延迟+持续稀释将导致投资者承受巨额损失。

**支撑要素**:

1. **比特币价格崩塌风险**: BTC已从$126K跌至$67K（-47%），JPMorgan估算当前挖矿成本$77K/BTC，已处于亏损边缘。若BTC跌至$40-50K，挖矿收入将腰斩，过渡期现金流断裂。

2. **执行风险巨大**: 从23K GPU扩展到140K GPU是6倍增长，Q2 FY26营收已miss 20%。Sweetwater建设可能延迟、GPU交付可能延迟、液冷系统调试可能延迟 -- 任一环节出问题都会形成"折旧和利息成本先于收入上升"的现金流真空期。

3. **股权稀释无底线**: 2年内流通股增长500%+，$2.3B可转债转股将再次稀释。公司至今无股利、无回购，所有现金流用于扩张。CEO在高位减持$66M后零买入 -- 最了解公司的人选择套现。

4. **AI泡沫破裂风险**: 如果AI投资回报未达预期，超大规模客户可能缩减资本开支。Microsoft合同虽为5年期，但具体违约金和延期条款未公开。

5. **客户集中度极高**: AI收入几乎100%来自Microsoft，单一客户依赖度在上市公司中极为罕见。任何合同变更/终止将对股价造成50%+的冲击。

6. **公司治理红旗**: 兄弟Co-CEO年薪各$72M（澳洲最高）、高位双减持$66M、CFO刚更换不到6个月、核心决策高度集权。

**Bear Case估值**: FY2029E 营收$2.0B x 净利率5% = 净利润$0.10B，EPS ~$0.20（500M稀释后），给予10x PE = **$2.00/股**。资产重置价值保底: **$8-12/股**。

## 十、风险提示与免责声明

### 10.1 完整风险清单

**高风险**:

1. **比特币价格崩塌** (概率: 中, 影响: 极高)
   - 描述: BTC已从$126K跌至$67K，当前挖矿成本$77K/BTC已接近亏损。若BTC进一步跌至$40-50K，挖矿收入将腰斩
   - 潜在影响: 综合估值下调28-40%，过渡期现金流断裂
   - 投资者应对: 设定BTC $50K为硬止损触发线

2. **GPU部署/数据中心建设延迟** (概率: 中, 影响: 极高)
   - 描述: 从23K到140K GPU是6倍增长，电网通电、GPU交付、液冷调试任一环节延迟都将导致"成本先于收入"的现金流真空
   - 潜在影响: AI业务价值下降30-40%，可能触发追加融资和稀释
   - 投资者应对: 每季度跟踪GPU部署数量和Sweetwater建设进度

3. **持续大规模股权稀释** (概率: 极高/确定性, 影响: 高)
   - 描述: 2.5年流通股增长4.5倍，$2.3B可转债未来将进一步转换，公司仍需额外$1-2B+融资
   - 潜在影响: 每股收益被持续摊薄，即使总收入增长也可能无法抵消稀释
   - 投资者应对: 使用稀释后股数(420-450M)进行估值

4. **客户集中风险** (概率: 中低, 影响: 极高)
   - 描述: AI收入几乎100%来自Microsoft单一客户，合同详细条款（违约金、延期条款等）未公开
   - 潜在影响: 合同变更/终止将导致股价50%+的冲击
   - 投资者应对: 关注Microsoft AI战略变化和新客户签约进展

5. **AI泡沫/资本开支周期性** (概率: 中低, 影响: 高)

   - 潜在影响: 新客户签约放缓，现有合同可能被延期

   - 潜在影响: 利息支出上升压缩本就微薄的利润率
