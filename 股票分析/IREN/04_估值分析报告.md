## 4. DCF 三情景分析

### 4.1 核心假设

#### WACC计算

```
E = 市值 = $13.26B
D = 净债务 = $0.58B（总债务$3.84B - 现金$3.26B）
  注: 使用净债务而非总债务，因现金充裕

Re (股权成本) = Rf + Beta × MRP
   = 4.3% (10年期美国国债) + 4.28 × 5.5% (美股MRP)
   = 4.3% + 23.5%
   = 27.8%

⚠️ 问题: Beta 4.28导致WACC极高(27.8%+)，使DCF几乎不可能得出有意义的正值。
这是因为:
1. IREN是加密/科技混合体，Beta反映了极端波动性
2. 高Beta是结构性特征，非暂时性
3. 但用27.8%的折现率对一个高成长基础设施公司估值明显过于惩罚

⚠️ 调整方案: 采用"行业调整Beta"方法
- 矿企行业Beta中值: ~2.5
- AI基础设施Beta中值: ~1.5
- 混合加权Beta: 2.5 × 0.5 + 1.5 × 0.5 = 2.0
- 调整后Re = 4.3% + 2.0 × 5.5% = 15.3%

Rd (债务成本) = 加权平均票面利率
   可转债: 0.25%($1.15B) + 1.00%($1.15B) + 定期贷款~5%($1.54B)
   加权Rd ≈ 2.5%
   税后Rd = 2.5% × (1 - 10%) = 2.25%（有效税率约10%，因亏损结转）

权重:
E/(E+D) = $13.26B / ($13.26B + $0.58B) = 95.8%
D/(E+D) = $0.58B / ($13.26B + $0.58B) = 4.2%

WACC = 95.8% × 15.3% + 4.2% × 2.25% = 14.66% + 0.09% = 14.75%

三情景WACC:
- 悲观: 17.0%（+2.25pp风险溢价，反映执行和BTC风险）
- 基准: 14.75%（标准计算）
- 乐观: 12.5%（-2.25pp，假设AI转型成功降低风险溢价）
```

**关键DCF前提**:
- IREN当前FCF为深度负值（FY2025: -$1.13B），DCF需要从"负FCF到正FCF"的拐点建模
- 预测期10年（FY2026-FY2035），分三个阶段：
  - 投资期（Year 1-3）: FCF持续为负，大规模CapEx
  - 收获期（Year 4-7）: AI收入放量，FCF转正并快速增长
  - 成熟期（Year 8-10）: 增长放缓，FCF稳定增长
- 永续增长率: 基准2.5%（≤10年期国债4.3%）
- **比特币价格假设**: 悲观$50K / 基准$90K / 乐观$130K
