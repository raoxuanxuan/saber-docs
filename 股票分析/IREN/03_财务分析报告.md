## 执行摘要

IREN Limited 正处于从比特币挖矿公司向综合数据中心/AI基础设施公司的激进转型期。FY2025（截至2025年6月）收入达$0.50B，同比增长163%，展现爆发式增长态势。然而，公司财务报表受比特币公允价值变动影响极为剧烈（Q1 FY2026录得$0.65B未实现收益，Q2则出现-$0.11B损失），导致GAAP利润波动极大，严重干扰对核心经营表现的判断。公司正以极高的资本支出（FY2025 CapEx $1.37B，为收入的2.7倍）大举扩建数据中心，导致自由现金流深度为负（FY2025 FCF -$1.13B），并不得不通过持续的大规模股权融资（Q2 FY2026单季增发$1.63B）来支撑扩张，股权稀释问题严峻。总体而言，IREN财务呈现"高增长、高投入、高杠杆、高波动"的特征，属于典型的基础设施扩张期公司，核心风险在于资本支出回报的不确定性和比特币价格波动对财务稳定性的冲击。

**杜邦分析 (TTM, 截至Q2 FY2026)**:
```
ROE = 净利率 x 资产周转率 x 权益乘数
    = (TTM净利润/TTM营收) x (TTM营收/平均总资产) x (平均总资产/平均股东权益)

TTM净利润: $0.39B (含大量BTC公允价值变动)
TTM营收: $0.76B
平均总资产: ($7.03B + $2.94B) / 2 = $4.99B
平均权益: ($2.51B + $1.82B) / 2 = $2.17B

ROE = ($0.39B/$0.76B) x ($0.76B/$4.99B) x ($4.99B/$2.17B)
    = 51.3% x 0.15 x 2.30
    = 17.7%

驱动因素分析:
- 净利率 51.3%: 极高（但不可持续，BTC公允价值驱动）
- 资产周转率 0.15: 极低（大量在建资产未产生收入）
- 权益乘数 2.30: 中等偏高（杠杆水平上升）
- ROE提升路径: 提升资产周转率（在建工程投产后）
```

**剔除BTC公允价值后的调整杜邦分析**:
```
调整后净利润(Normalized): ~$0.05B
调整ROE = ($0.05B/$0.76B) x ($0.76B/$4.99B) x ($4.99B/$2.17B)
        = 6.6% x 0.15 x 2.30
        = 2.3%

结论: 核心经营ROE仅约2.3%，远低于资本成本，反映当前产能利用率低和扩张期投入大
```

## 8. 财务分析结论

### 财务优势 (Top 3)

1. **爆发式收入增长**: FY2025收入$0.50B同比增长163%，3年CAGR~103%。AI/HPC数据中心战略布局为未来收入增长提供了明确路径，目标510MW产能较当前规模有数倍增长空间。

2. **高毛利率与经营现金流改善**: 毛利率从FY2023的25%提升至68%，经营现金流从FY2022的-$0.04B改善至FY2025的$0.25B。核心挖矿业务已实现正向现金流贡献。

3. **短期流动性极为充裕**: Q2 FY2026后持有$3.26B现金，流动比率4.97，短期偿债无任何压力。大规模融资确保了未来12-18个月的建设资金需求。

### 财务风险 (Top 3)

1. **比特币价格波动对财务的致命性影响**: GAAP利润完全被比特币公允价值变动主导（Q1 FY2026单季$0.65B收益，Q2 -$0.11B损失）。若比特币价格大幅下跌（如跌至$40K以下），将同时冲击挖矿收入和资产价值，可能触发债务违约。

2. **持续大规模股权稀释**: 2.5年内流通股增加4.5倍（0.06B->0.33B），累计股权融资$4.67B。在核心经营利润微薄（Normalized净利润仅$0.02B/年）的情况下持续增发，严重损害每股权益。未来仍有继续融资需求。

3. **自由现金流深度为负且CapEx回报不确定**: FCF连续4年为负（FY2025: -$1.13B），CapEx为营收的2.7倍。数十亿美元的数据中心投资能否产生预期回报取决于AI算力需求是否如期兑现，以及IREN能否获得优质长期客户合同。任何延期或需求不及预期都将恶化财务状况。
