## 1. DCF 绝对估值

### 1.1 核心假设

**宏观参数:**
- 无风险利率 (Rf): 4.08% (10Y美债, 2026-02-18)
- 市场风险溢价 (MRP): 5.5%
- Beta: 2.47 (当前) / 1.80 (标准化, 剔除近期异常波动)
- 永续增长率 (g): 3.5% (< 10Y美债收益率4.08%)

**公司特定参数:**
- FY2025 FCF估算: ~$250M (FCF Margin 10.7%)
- FY2024 FCF: $209M
- 历史营收CAGR (2022-2025): ~65%
- 净利率: ~5% (FY2025E)
- 摊薄股数: 228M (当前) / 247M (含Eucalyptus稀释)

### 1.2 WACC 计算

```
Re = Rf + Beta(标准化) x MRP
   = 4.08% + 1.80 x 5.5%
   = 4.08% + 9.90%
   = 13.98%

由于公司当前净现金为正, WACC ≈ Re
考虑GLP-1监管风险溢价 (+0.5pp): WACC = 14.5%

WACC范围:
- 悲观: 15.5% (+1pp 风险溢价)
- 基准: 14.0%
- 乐观: 12.5% (-1.5pp, 风险消退)
```

**WACC合理性检验**: 中盘成长股典型区间9-12%, 考虑当前GLP-1+诉讼风险, 14%处于偏高但合理范围。
